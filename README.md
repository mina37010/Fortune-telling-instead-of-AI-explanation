# Fortune-telling-instead-of-AI-explanation
選択したテキストをワンクリックで /tanka(https://いなにわうどん.みんな/tanka) に投稿できる Chrome 拡張です。
投稿者名も設定すれば自動で入力されます。
いなにわうどん許諾済み 感謝

## 機能

- ChromeのAI要約を占いに変えます

## インストール方法

1. このリポジトリをクローンまたはZIPダウンロード
2. Chromeで `chrome://extensions/` を開く
3. 「デベロッパーモード」をONにする
4. 「パッケージ化されていない拡張機能を読み込む」→ このフォルダを選択
